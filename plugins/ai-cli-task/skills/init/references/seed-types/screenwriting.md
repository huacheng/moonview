# screenwriting

## Description

Film/TV scripts, storyboards, dialogue

## Methodology

Format compliance, scene structure, dialogue flow

## Phase Intelligence

### plan

- **Collection Direction**: Industry format standards, story structure, production constraints
- **Key Sources**: Final Draft/Fountain spec, Syd Field's paradigm, production company formatting guides
- **Plan Structure**: Logline → treatment → beat sheet → scene breakdown → draft
- **Key Considerations**: Industry format (Final Draft/Fountain), three-act structure, dialogue rhythm, visual storytelling

### verify

- **Collection Direction**: Format validators, continuity checkers, script analysis
- **Key Sources**: afterwriting, Fountain.js, script coverage services, industry format specs
- **Quick Checkpoint**: Fountain/FDX format valid, scene count matches outline
- **Full Checkpoint**: Industry format compliance, scene heading format, dialogue balance, continuity notes cross-check
- **Key Tools**: `afterwriting` (Fountain linter), `fountain-js`, custom format validators, page count check

### check

- **Collection Direction**: Industry format standards, story structure quality, production feasibility
- **Key Sources**: WGA standards, coverage service criteria, production budget constraints
- **Indicators**: Film/TV script, storyboard, dialogue
- **Verification Approach**: Script format compliance (industry standard), scene structure, dialogue flow, continuity checks

### exec

- **Collection Direction**: Fountain/FDX syntax, scene writing techniques, production annotation formats
- **Key Sources**: Fountain spec, screenwriting software docs, production notation standards
- **Implementation Approach**: Write scenes in industry format (Fountain/Final Draft), maintain continuity notes
- **Step Verification**: Format compliance, scene structure validation, dialogue flow, continuity checks
